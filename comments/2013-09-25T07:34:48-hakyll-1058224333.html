Ну не то чтобы системой, ведь Hakyll собирает все, что нужно. А чтобы не запускать вручную используется Github Webhook. Как только приходит push на Github, он дергает заданный url с информацией что пришло. На сервере при запросе на этот урл у меня стоит вызова git pull && ./dikmax-name build (все можно можно глянуть в исходниках: update.sh и src/Server.hs)
